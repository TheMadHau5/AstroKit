## Prerequisites
- [NodeJS](https://nodejs.org) v16+
- [Python](https://www.python.org/) v3.11+

## Installation 
1. Install packages
```bash
npm install
```
2. Initialize CSS
```bash
npm run start
```
3. Run the app!
```bash
py main.py
```
